Proyectos realizados

  1. Sistema Contable

    2. En este proyecto se realizó una página web con diseño minimalista utilizando las tecnologías de XAMPP, PHP y MySQL donde se registran las partidas contables mostrando un balance general y un estado de resultados, además de mostrar los diferentes libros contables con tablas que facilitan el entendimiento del mismo. Todo se almacena en una base de datos local.
